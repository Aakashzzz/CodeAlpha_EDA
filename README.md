# Exploratory Data Analysis (EDA) Using Python

## Overview
This project demonstrates Exploratory Data Analysis (EDA) using Python on an employee dataset. The objective is to understand the dataset structure, identify patterns, detect anomalies, and visualize relationships between variables.

## Dataset
The dataset contains the following columns:
- Name
- Age
- Gender
- Salary

## Analysis Performed
- Data Loading
- Data Inspection
- Dataset Shape Analysis
- Column Identification
- Statistical Summary
- Missing Value Detection
- Duplicate Record Detection
- Histogram Visualization
- Correlation Heatmap
- Box Plot Analysis
- Scatter Plot Analysis

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Output
The project generates:
- Statistical summaries
- Data quality checks
- Histograms
- Correlation Heatmaps
- Box Plots
- Scatter Plots

## Learning Outcomes
- Understanding dataset structures
- Performing statistical analysis
- Detecting data quality issues
- Creating meaningful visualizations
- Drawing insights from data
